# Contributor License Agreement（CLA）

为支持 `zhenyi` 以 **AGPL-3.0** 开源并同时提供商业授权，你在向本项目提交贡献（代码、文档、测试、设计等）前，需要同意以下条款。

> 本 CLA 用于明确贡献来源、再许可权利与双授权边界，作为本仓库当前公开适用的贡献协议文本。

## 1. 定义

- “你/贡献者”：向本项目提交贡献的个人或实体。
- “贡献”：你提交并被项目接受的任何内容（包括但不限于代码、文档、测试、示例、配置）。
- “项目方”：`zhenyi` 仓库的权利管理方（仓库所有者及其明确授权的维护者）。

## 2. 贡献者声明

你声明并保证：

- 你有权提交该贡献，且该贡献不侵犯任何第三方权利；
- 如贡献包含第三方内容，你已获得合法授权并在 PR 中明确标注来源与许可；
- 你理解并接受本项目为双授权模式（AGPL + 商业授权）。

## 3. 授权条款（核心）

你在此向项目方授予一项**永久的、全球范围的、不可撤销的、非独占的、免版税的**许可，使项目方可以：

- 以 **AGPL-3.0** 许可方式使用、复制、修改、分发并再许可你的贡献；以及
- 以项目方提供的**商业授权条款**使用、复制、修改、分发并再许可你的贡献（包括向第三方授予商业授权）。

该授权允许项目方在不另行征得你同意的情况下，为了项目整体一致性而对贡献进行合并、拆分、重写、格式化等必要处理。

## 4. 同意方式与生效

- 你在 GitHub/Git 平台提交 Pull Request（或其他形式提交并请求合并）即视为你已阅读并同意本 CLA。
- 本 CLA 对你被项目接受（合并）的全部贡献生效。

## 5. 专利许可（如适用）

如你的贡献包含你/你所属实体拥有或可控制的专利权，你同意向项目方及其被授权方授予一项**非独占、全球范围、免版税、不可撤销**的专利许可，以实施、使用、销售、分发包含该贡献的实现。

## 6. 无报酬

除非双方另有书面约定，你的贡献不产生任何报酬或对价。

## 7. 署名与归属

项目方可在合理范围内保留你的署名信息（例如 Git 提交记录、贡献者列表）。你同意项目方可为合规与发布目的整理贡献者信息。

## 8. 适用法律与争议解决

如发生争议，双方应先友好协商解决；协商不成时，争议提交项目方所在地有管辖权的法院处理。  
如双方另有书面约定，以该书面约定为准。
